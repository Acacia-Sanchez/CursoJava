<H1 align=center>CursoJava</H1>
<h3 align=center>EJERCICIOS "SENCILLOS" PARA PRACTICAR JAVA</h3>

**Variables**
1. Declarar e inicializar variables: Escribe un programa que declare e inicialice
diferentes tipos de variables (int, float, char, etc.), luego imprímelas.
2. Intercambiar dos variables: Escribe un programa que intercambie dos
variables enteras sin usar una tercera variable.

**Intercambiar Dos Variables**

3. Intercambiar usando operaciones aritméticas: Modifica el ejercicio anterior
para intercambiar dos enteros usando operaciones aritméticas (por ejemplo,
suma y resta).
4. Intercambiar usando XOR bit a bit: Intercambia dos variables enteras usando
la operación bit a bit XOR.

**Entrada de Usuario**

5. Leer y mostrar entrada del usuario: Escribe un programa que lea un entero,
un número decimal y una cadena de texto ingresados por el usuario, luego
muéstralos.

6. Suma de dos números: Escribe un programa que lea dos números enteros del
usuario e imprima su suma.

**Expresiones**

7. Expresiones aritméticas simples: Crea un programa que evalúe una expresión
aritmética simple ingresada por el usuario (por ejemplo, 3 + 5 * 2).

8. Calcular área de un círculo: Escribe un programa que calcule el área de un
círculo en función del radio ingresado por el usuario.

**Introducción a GUI**

9. Aplicación GUI simple: Crea una interfaz gráfica básica con JOptionPane que
le pida al usuario su nombre y luego muestre un saludo.

10. Calculadora GUI: Escribe una calculadora simple con interfaz gráfica usando
JOptionPane que pueda sumar, restar, multiplicar y dividir dos números.

**Clase Math**

11. Operaciones matemáticas: Escribe un programa que demuestre el uso de
Math.pow(), Math.sqrt(), y Math.abs().

12. Generar números aleatorios: Escribe un programa que genere e imprima un
número entero aleatorio entre 1 y 100 usando la clase Math.

**Números Aleatorios**

13. Simulador de tirada de dados: Crea un programa que simule la tirada de un
par de dados usando números aleatorios.

14. Generador de contraseñas aleatorias: Escribe un programa que genere una
contraseña aleatoria de 8 caracteres usando letras y dígitos.

**Sentencias If**

15. Verificar si un número es par o impar: Escribe un programa que lea un
número del usuario y verifique si es par o impar.

16. Comprobar si un año es bisiesto: Escribe un programa para verificar si un año
dado es bisiesto usando una sentencia if-else.

**Sentencias Switch**

17. Calculadora simple usando switch: Escribe un programa que realice suma,
resta, multiplicación o división según la entrada del usuario usando una
sentencia switch.

18. Días de la semana: Escribe un programa que convierta un número (1-7) en el
día correspondiente de la semana usando una sentencia switch.

**Operadores Lógicos**

20. Verificación de elegibilidad para votar: Escribe un programa para comprobar
si un usuario es elegible para votar (edad &gt;= 18) usando operadores lógicos.

21. Condiciones lógicas: Crea un programa que verifique si un número es positivo
y par usando AND lógico, y si es negativo o divisible por 3 usando OR lógico.

**Bucles While**

21. Suma de números naturales: Escribe un programa que calcule la suma de
números naturales hasta un número dado usando un bucle while.

22. Juego de adivinar el número: Escribe un juego donde el usuario debe adivinar
un número generado aleatoriamente entre 1 y 100.

**Bucles For**

23. Imprimir tabla de multiplicar: Escribe un programa que imprima la tabla de
multiplicar de un número ingresado por el usuario usando un bucle for.

24. Cálculo de factorial: Escribe un programa que calcule el factorial de un número
usando un bucle for.

**Bucles Anidados**

25. Imprimir patrones: Escribe un programa para imprimir un patrón piramidal de
estrellas usando bucles anidados.

26. Multiplicación de matrices: Escribe un programa que multiplique dos matrices
ingresadas por el usuario usando bucles anidados.

**Arrays**
27. Encontrar el mayor elemento en un array: Escribe un programa que
encuentre e imprima el mayor elemento en un array de enteros.

28. Invertir un array: Escribe un programa que invierta un array de enteros.

**Arrays 2D**

29. Suma de matrices: Escribe un programa que sume dos arrays 2D y muestre el
resultado.

30. Transpuesta de una matriz: Escribe un programa que encuentre e imprima la
transpuesta de un array 2D.

**Métodos de String**

31. Palíndromo de cadena: Escribe un programa que verifique si una cadena dada
es un palíndromo usando métodos de String.

32. Contar vocales en una cadena: Escribe un programa que cuente el número de
vocales en una cadena ingresada por el usuario.

**Clases Wrapper**
33. Convertir cadena a entero: Escribe un programa que convierta una cadena de
texto a un número entero usando clases envolventes y le sume 10.

34. Valores máximo y mínimo de tipos: Escribe un programa que imprima los
valores máximo y mínimo de int, double, y char usando métodos de las clases
envolventes.

**ArrayList**

35. Lista dinámica de enteros: Escribe un programa que permita al usuario
ingresar enteros en un ArrayList, luego muestre la lista y su suma.

36. Eliminar duplicados: Escribe un programa que elimine los elementos
duplicados de un ArrayList de cadenas de texto.
